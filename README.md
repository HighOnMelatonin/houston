## Getting Started
### Dependencies
Have python3 installed, all dependencies needed will be installed in a local virtual environment in the `src` directory, just follow [execution](#execution)

### Hardware Description
A Raspberry pi 4

## Execution
Run `start.sh` by typing ```./start.sh``` at the root of this repo

## Folder Structure
'src' contains the source code, along with the virtual environment and the python libraries needed for the main program

## Code Documentation
The main dependency of the program is [gpiozero](https://gpiozero.readthedocs.io/en/stable/recipes.html)
